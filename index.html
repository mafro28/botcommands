mafrobot commands



mafrobot is a Twitch chat bot and moderator for mafroafro's channel that runs in the cloud. Below are the chat commands:



!blm - a link to how you can support Black Lives Matter

!age - show how long mafroafro has been on Twitch.

!followage - show how long the user has followed mafroafro.

!game - show the current game mafroafro is playing.

!lurk - use to show that you're watching, but lurking in chat.

!playtime - show how long mafroafro has been playing the current game.

!deathctr - show the current amount of total deaths so far.

!prime - an Amazon Prime sub reminder ðŸ¥º

!socials - show links to mafroafro's Twitter and Instagram.

!title - show the current title of the stream.

!random - produces a random message.

!slots - play the slot machine game.icon

!uptime - show how long the current stream has been live for.

!xmas - countdown timer for Christmas day.

!roulette - play Russian Roulette!

!adventure (points) - start an adventure with a set amount of points.

!8ball - ask the magic 8ball a question.
